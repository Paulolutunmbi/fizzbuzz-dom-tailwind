# FizzBuzz (DOM Version)

A simple FizzBuzz implementation that runs in the browser and displays results on the page instead of the console.

## What this project demonstrates
- JavaScript loops and conditionals
- DOM manipulation
- Basic HTML structure
- Styling with Tailwind CSS (CDN)

## How it works
The script loops from 1 to 50 and:
- Displays "FizzBuzz" for numbers divisible by both 3 and 5
- Displays "Fizz" for numbers divisible by 3
- Displays "Buzz" for numbers divisible by 5
- Displays a default message otherwise

## Technologies used
- HTML
- JavaScript
- Tailwind CSS

## Purpose
This project was built as a learning exercise to practice JavaScript logic and DOM rendering.
